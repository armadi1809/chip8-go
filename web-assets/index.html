This is a chip 8 emulator written in golang and ebitengine to handle graphics, audio and keyboard input. You can change the ROM from the dropdown in the top left of the emulator screen

Chip 8 pad to keyboard mapping:

1 -> 1

2 -> 2

3 -> 3

C -> 4

4 -> Q

5 -> W

6 -> E

D -> R

7 -> A

8 -> S

9 -> D

E -> F

A -> Z

0 -> X

B -> C

F -> v
